## ChatGPT 官网注册与使用教程

1. **访问官网**：[https://chat.openai.com](https://chat.openai.com)（需翻墙）。
2. **注册账号**：
   - 需准备一个海外手机号进行验证。
   - 填写邮箱，设置密码完成注册。
3. **选择版本**：登录后选择 GPT-4 或 GPT-3.5。
4. **开始对话**：输入各种问题，享受智能回答。
